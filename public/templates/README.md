# Document Templates

This directory contains downloadable document templates for vendors.

## Available Templates

- `bank-letter-template.pdf` - Bank letter template for bank details change requests
- `tax-certificate-template.pdf` - Tax certificate template for compliance
- `compliance-doc-template.pdf` - General compliance document template

## Usage

Templates are served via the `/templates/:filename` route and can be downloaded from the upload guidance section in the case evidence panel.
